# Claude Plugins

Custom Claude Code plugin marketplace by SugaredPlacebo.

## Plugins

| Plugin | Description |
|--------|-------------|
| **reactflow** | ReactFlow (@xyflow/react) patterns, rules, and references for writing correct node-based UI code |

## Installation

Add this marketplace to Claude Code:

```
SugaredPlacebo/claude-plugins
```

Then install any plugin:

```
claude plugin install reactflow
```
